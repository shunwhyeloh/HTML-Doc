1. Web

1.1. What is Web?
-Web is invented in 1990s and it contains of 30 billion pages.
-Web is a series of HTML (Hyper Text Markup Language) documents.

1.1.1. Web Major concepts
For the process of web navigation, there are 3 main roles which are the user with the browsers, the internet, and the servers.
It also consists of 4 stages during the process:
a) Users use browsers (e.g. Chrome, FireFox, etc.) to request internet to search for a specific information.
b) The search request will be delivered through the internet to the servers ("storages" that contain extensive amount of web pages), using the protocol of http.
c) The servers will then request the internet to respond to the user's search request.
d) Information respond base on the search request will then be displayed on user's digital gadget.

2. HTML

2.1. What is HTML?
-HTML is the most fundamental type of document for all web pages that helps to link everything together.
-The link that helps to connect all the media contents called the Hyper Link.

2.1.1. HTML Tags & Elements
-HTML markup is made up of tags.
-Tag that has no content such as image does not need a closing tag and this is known as void tag.
-The whole line that is created with tags called element.

2.1.2. HTML Attributes
-HTML Attrributes serve to add value for tags.

2.2. Inline vs Block
br= inline | p= block
-br creates a space between words or to separate the one line into two separate lines.
-p creates an invisble box with heigh and width around a text/sentence. While br simply helps to break one line into two and create space between words.
-Basically, the diffrence between br and p is the amount of space that they can create.
-Inline tags: b, br, img, em, span, strong
-Block tags: h, t, p, div, form

2.3. Span & Div
-Span and Div both have content.
-The difference between Span and Div is that Span is incline and Div is block.
